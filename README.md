# Generative UI Demo

Uses a TUI to demo generating structured content that can be used to render dynamic UIs.

## Setup

This example uses Azure OpenAI to generate the UI Response.

It can be easily modified to use another provider from the list here: https://ai-sdk.dev/providers/ai-sdk-providers

The environment variables can be configured by setting them in the `.env` file.

## Running

To run the applications TUI, run the command `npx tsx src/cli.tsx`.
